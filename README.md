# Browser-Extensions
To-do List , Grammer checker browser extension 

Simple second sem HTML Frontend project based on API integration .

A To-Do List Browser Extension is a productivity tool designed to help users manage daily tasks directly from their browser. 
It allows users to quickly add, edit, and delete tasks without opening a separate application. 
The extension stores data locally using browser storage APIs, ensuring tasks remain available even after the browser is closed.
When the user opens the extension popup, they can type a task and add it to the list. Each task can be marked as completed, helping users track progress efficiently. The interface is simple, fast, and optimized for quick interactions. Modern extensions may also include features like due dates, daily reminders, color-coded tasks, and synchronization across devices using cloud storage.

This extension is highly useful for students, professionals, and anyone who wants to stay organized while working online. Because it runs inside the browser, it improves productivity by keeping tasks accessible at all times, right next to the userâ€™s browsing activities.

A Grammar Checker Browser Extension is an AI-assisted tool that helps users improve writing quality by detecting and correcting grammar, spelling, punctuation, and sentence structure errors in real time. The extension integrates directly with the browser and works on any text input field, such as emails, social media posts, blogs, or online documents.

The extension continuously monitors the text typed by the user. Using NLP (Natural Language Processing) techniques and machine learning models, it identifies grammar mistakes and suggests corrections. It may highlight errors, show explanations, and offer alternative phrasing. The extension uses background scripts to analyze text and popup windows to display suggestions and configuration settings.

It uses a local Grammer Checking API easily Available on the internet .
